# Cotton Disease Detection

## Project Overview
This project uses **Convolutional Neural Networks (CNNs)** to classify images of cotton leaves and plants into **diseased** or **fresh** categories. The model is based on **VGG16** and can be trained and evaluated on GPU for faster performance.

---

## Dataset
The dataset used is from [Kaggle: Disease Cotton](https://www.kaggle.com/ananysharma/diseasecotton).  
It contains images of:

- Fresh cotton leaf  
- Fresh cotton plant  
- Diseased cotton leaf  
- Diseased cotton plant
